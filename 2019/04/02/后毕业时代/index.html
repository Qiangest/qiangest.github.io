后毕业时代

发表于 2019-04-02 |
字数统计: 1,828

每次写博文的时候我都会感慨，为什么总觉得这一个月才刚开始，突然就结束了呢？但是回过头来再想想，好像这个月发生的事情又的确是很遥远了。

2019年已经过了四分之一，现在是在硕士最后一学期的尾巴上。现在的生活比较清闲，人也变得懒了。自从老赵走后，组内的锻炼氛围日益减少，现在不打球也不健身了，想他。现在组会基本上也没有什么任务，多是参与，主要工作转变为处理遗留下来的科研问题，自己看看最近的文章，学习一些深度学习的知识。其实在最后的三个月，对自己的要求还是有的：

  • 把两篇文章extend一下投journal

  • 辅导本科和硕士师弟的毕业设计，activity项目基本进入实验阶段；

  • 自己也想一些idea做个初步的survey等博士的时候直接上手做

  • 还有就是把落下的机器学习、深度学习、信号处理的部分再学一学，系统性的梳理一下

  • 还有就是把自己一团糟的生活好好整理一番；

  • 学好英语

    这样看来其实任务并不轻松呀。

3月初把论文又重新加了一些内容，达到了70页左右。现在毕业论文就是这样，虽然字数早就够了，但是大家都写很多，你写的薄了话就会fail掉。格式和厚度已经快是答辩的第一标准了。查重结果0.5%，看到一些重复的内容也是一些专业术语，不打算在这个上面花费太多时间了。前几天ICDCS的结果出来了，EchoWrite这个项目终于算是扔了出去，不知道赶过多少Deadline，被reject多少次。最终还是没有能中A，也明显感觉到A与B、ACM与IEEE的区别，自己也的确还需要修炼好内功，做出idea新颖、方法深刻、实现solid、实验详尽的好work来。继续加油吧！这次在Dallas开会，不知道还可以不可以去，自己尽量争取吧。好在发表后答辩的底气就更强了。

中旬去日本开会，多出去走一走还是可以开阔眼界的。在陌生的国度说英语就觉得很自然，怪不得留学对英语提高很大。可以看出日本受中国的文化影响还是很大的，地名等实意名词什么的都是中文，也不会有太多障碍。在京都传统和现代结合得很好，没有中国这些城市的突兀感，更没有那种不古不新莫名其妙的感觉。或许是因为人少，最宽的马路都没有学校旁边的路宽，也没见会有堵车，所以所有的公交车都是按时刻表精确到分钟到达的，这个让我很吃惊。在中国这几乎是不可能的。也正因为如此所有的地铁和电车都是按时刻表发车，中间间隔还挺久的。在中国的话这么搞不一会地铁非挤爆不可。值得借鉴的是日本的公共交通是真的发达，巴士、地铁、火车、电车在城市中四通八达，几乎可以去任何想去的地方。只需要一个ICOCA（类似八达通)，换乘都很方便。这样大家对私家车的需求就会大大降低，城市的拥堵也会少很多。但是发达国家的车费是真的很贵，做一次车基本上都在人民币25块以上。

京都的农村整洁而干净，房子虽然很低但是很有感觉，不像中国农村房子摞房子，毫无空间设计。过了9点基本上所有的商店都打烊了，或许商店关门的时间也可以衡量人民的生活水平？整体来看中国发展国家的定位还是很实事求是的，还有很多空间需要改进。另外还有一个点，三月其实还算是初春，寒风还是很刺骨的，但是日本的女学生大都是上面羽绒服下面短裙光腿，真的是不怕冷吗？虽然在日本的中国人和韩国人都很多，但是还是可以看出他们的明显区别的，日本男人白领大都是衬衣领带加大衣，手里提个公文包，面无表情的…这次的遗憾就是樱花都还没开，很多地方却了樱花就变成秃秃的一片，毫无美感可言了。

月末F4家族一起约了一趟桂林，相比上一次去顺德来说已经非常成功了。给我的感觉是商业气息很重，很多是个人的旅游开发，也毫无审美可言。感谢高铁，两个小时内可以到这么远的地方。我更喜欢的是到一个陌生的地方，好好的睡一觉，吃点东西，在风景秀丽的地方随性的转一转，让自己放松一下。这次全程都在录vlog，回过头看真的觉得很好笑，也很有意思，以后打算再剪一剪，也可以当做科研素材做一下STT转换，或者人脸识别。人熟悉了后开玩笑也变得无下限，这些也引发了我很多关于人性的思考，也挺有意思的。每次旅行希望都可以带来新的思考和体验。如果真的只是看看风景倒对我没有什么吸引力了。有时候的确需要一段时间来整理自己的思绪，不然很容易埋没在regular的生活中：重复最容易让人失去思考的能力。那么跳出来的方法其实很多人都了解，只是没有意识到：阅读，和旅行。旅行要收到金钱限制，那么还是多多阅读吧。

本科最后一个学期的几个月是我最快乐的日子，希望硕士也可以如此。这段时间研三的人几乎都不在实验室了，就我还每天坚持来9/10/6地工作，简直要比Github上那些被控诉的企业还让人感动。因为我知道我还是有很多事情没有完成，只是自由是珍贵的，但真正有了自由却不见得可以利用好，没有任务严格约束的自己也是缺乏动力和干劲的，自己也因此常常效率变低，心态浮躁，对自己的要求标准越来越松懈。所以这段时间也是一个修炼的过程，如何在自由中更自律，保持干劲沉下去做事。如何持续学习，接触新事物，补齐自己的知识结构，如何利用自由的时间做自己喜欢的事，如何刻意地训练自己的身体、心理与控制力，对我来说都是一个持续的问题。让自己的生活充实，身体健康，Keep moving!!!
